# 排序法介紹
## 氣泡排序法
基本原理
在排序的過程中，數值較大（或較小）的元素會透過不斷的交換，比較相鄰的兩個元素，並把大的往右邊推，所以一輪比較過後，當前陣列最大的元素一定會在最右邊
操作分析
第一回合（Pass 1）：從陣列的第一個元素開始，與第二個元素比較。如果第一個大於第二個，就交換它們。接著比較第二和第三個，依此類推，直到比較到陣列的最後一個元素。此時，全陣列最大的數字已經被安置在最後一個位置。

第二回合（Pass 2）：重複上述動作，從頭開始兩兩比較。但因為最後一個元素已經是最大值（已固定），所以這次只需要比較到倒數第二個位置。結束後，第二大的數字會被安置在倒數第二個位置。

重複執行：每一回合都會讓一個剩餘的最大值歸位，因此每一回合需要比較的次數都會比上一回合少一次。

結束條件：當經過一整輪的比較，完全沒有發生任何交換動作時，就代表所有元素都已經在正確的位置上，排序完成
## 插入排序法
基本原理
1.初始狀態：將陣列的第0個元素視為「已排序區」，第1到第n-1個元素視為「未排序區」。
2.取出目標（Target）：從未排序區取出第一個元素作為「目標值」，準備將它插入已排序區。
3.由後向前比較：將目標值與已排序區的元素（從最右邊最大的開始）逐一比較：如果已排序區的元素大於目標值，就把該元素往右移動一格，騰出空間。如果已排序區的元素小於或等於目標值，或者已經比對到陣列的最左邊（索引為 0），就停止比較。
執行插入：將目標值放入最後騰出來的那個空位中。此時，已排序區的長度增加1，未排序區減少1。
重複執行：重複步驟 2 到 4，直到未排序區沒有任何元素為止。
